# FlipbookEngine Trademark Policy

This policy governs use of the FlipbookEngine name, logos, and brand assets.

## Covered Marks

The following are treated as protected brand assets:

- FlipbookEngine
- FlipbookEngine logos
- any official product lockups, icons, and visual identity files distributed with the project

## What the Open Source License Does Not Grant

The AGPL license applies to source code, not to trademark rights.

Using the FlipbookEngine code does not automatically grant permission to:

- present your product as the official FlipbookEngine product
- use FlipbookEngine logos in your own branding
- imply sponsorship, certification, or endorsement
- register confusingly similar names, domains, or product marks

## Allowed Uses

Unless otherwise stated, you may:

- refer to FlipbookEngine by name in factual descriptions
- state that your project uses or is powered by FlipbookEngine
- link to the official project and documentation

## Uses That Require Permission

You must get permission before:

- using official logos in a product UI or marketing page
- using FlipbookEngine as part of your own product or company name
- presenting your fork as the official project
- using the marks in paid advertising or marketplace listings

## Open Source Attribution

Attribution is appreciated in open source projects, but trademark permission is still separate from code licensing.
