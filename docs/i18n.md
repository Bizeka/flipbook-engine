# Internationalization (i18n)

FlipbookEngine features built-in support for multiple locales and customization of all user-facing copy.

## Supported Locales

- **`en`** (English - default)
- **`tr`** (Turkish)

## Set Active Locale

Change the language on initialization or dynamically at runtime:

```javascript
// Initialization
const engine = new FlipbookEngine('#viewer', {
  locale: 'tr'
});

// Runtime change
engine.setLocale('en');
```

## Overriding Messages

Provide translation strings to override specific labels or define messages for a new locale:

```javascript
const engine = new FlipbookEngine('#viewer', {
  locale: 'de',
  messages: {
    de: {
      emptyStateTitle: 'Keine Seiten gefunden.',
      thumbnails: 'Miniaturansichten',
      singlePageMode: 'Einzelseiten-Modus',
      pages: 'Seiten',
      previous: 'Zurück',
      next: 'Weiter',
      zoomOut: 'Herauszoomen',
      zoomIn: 'Herinzoomen',
      fullscreen: 'Vollbild',
      settings: 'Einstellungen',
      downloadCatalog: 'PDF Herunterladen'
    }
  }
});
```
